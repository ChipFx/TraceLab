ChipFX TraceLab Help
=====================

TraceLab is a free-to-use Python based data visualiser in an oscilloscope jacket.

In fact, it is more than free-to-use, it comes with a modified Apache 2.0 license,
meaning that as long as you attribute whatever you build with this to ChipFX and 
the listed authors and you include the NOTICE.md contents in your project you 
can use it however you like in whatever manner you like.

We do also kindly request that you keep the branding, to give the project traction.
But we cannot "demand" you do under the base implications of Apache 2.0.

Please check the **Help → About TraceLab** section from the Menu Bar, or read the
LICENSE.md and NOTICE.md in the project's root.
