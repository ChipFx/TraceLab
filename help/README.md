# TraceLab Help Content

This folder holds all help topics shown in the TraceLab built-in help browser.

Each `.RST` file in this directory (and any sub-folders) is a help topic.
Topics are discovered automatically at startup by `HelpRegistry`; no
registration step is needed.

## File format

Topics follow the pyhelp RST dialect — see
[../pyhelp/README.md](../pyhelp/README.md) for the full authoring guide,
including front-matter fields, admonition syntax, and how topic ordering works.

## Viewing help in TraceLab

Open **Help → Help Contents** from the main menu bar.
