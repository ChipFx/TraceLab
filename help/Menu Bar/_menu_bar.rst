Menu Bar
===========

The Menu Bar, like on most other serious programs, allows you to perform
a great many tasks, operations and settings changes. In the chapters below
you can read a short description of each element and what can be found there.

For each menu there is a separate help subject in the main chapter `Menu Bar <rst-doc://menu_bar>`_.

File
-----------

The File menu allows you to open files, export the current view, take screenshots and clear all traces.
For more information see the `File Menu <rst-doc://menu_bar/file_menu`>_.

View
-----------

The View menu allows you to select between Split-Lane view and Overlay All Traces.
It has controls to zoom and fit the trace data and allows you to lock the Y axes to auto-scale,
meaning the Y axis auto-zooms on the highest peak-to-peak in the current view.
For these subjects see the `View Menu <rst-doc://menu_bar/view_menu`>_.

In the View menu you can also set the Interpolation mode for all traces, see `View: Interpolation <rst-doc://view_menu/interpolation`>_.

Or you can set the Draw Mode, see `View: Draw Mode <rst-doc://view_menu/draw_mode`>_.

And you can also select themes, for more about themes see `Settings: Themes <rst-doc://settings_menu/edit_theme`>_.

Analysis
-----------

The Analysis menu allows you to open the FFT window, see `Analysis: FFT <rst-doc://fft_window`>_.
It also allows access to the filter window `Analysis: Filter <rst-doc://filter_window`>_.

Apart from those you get access to a Clear Filters quick action, add a label at the A cursor,
clear the labels, see `Analysis Menu <rst-doc://menu_bar/analysis_menu`>_, and set the Interpolation 
per Channel. See `View: Interpolation <rst-doc://view_menu/interpolation`>_.


Acquire
-----------

The Acquire menu houses the main fancy-pants Oscilloscope trickery.
Allowing you to choose between no advanced acquisition (off), two types of persistence,
averaging and interpolation, as well as controls to change their detailed behaviour.

For more detailed inforamtion see `Acquire Menu <rst-doc://menu_bar/acquire_menu`>_.

Settings
-----------

The Settings menu houses all the settings and tweaks that did not belong in a dedicated sub-menu.
Mostly these settings are about basic program behaviour or UI/UX. It allows setting Lane Label settings,
for details about theme override on the labels and on the Theme Editor and Themes themselves, 
see `Settings: Themes <rst-doc://settings_menu/edit_theme`>_.

For more information about the other settings in the Settings menu see `Settings Menu <rst-doc://menu_bar/settings_menu`>_.

Plugins
-----------

The Plugins menu allows you to reload the plugins and open the plugins folder. The rest of the menu
will be populated by the plugins as they exist in the plugins folder.

For more information about plugins and how they are grouped into submenus, see `Plugins Menu <rst-doc://menu_bar/pluginss_menu`>_.

Help
-----------

Houses the help and about sections. See the `Help Menu <rst-doc://menu_bar/help_menu`>_.
