INCEPTION! Help Menu
======================

Help (F1)
----------------

Shows this help window using the ChpiFX/pyhelp.git project.

Press F1 to open help on any element in the program, or choose **Help → Help (F1)**.

About TraceLab
----------------

Shows a window with a small about section, followed by the License and
Notice file contents, to fully inform you of your rights and requirements when
using, copying or distributing this application.

Choose **Help → About TraceLab**
